## 一、产品定位

**避风港 SafeHarbor** 是一款面向中小跨境电商卖家的 AI 合规智能体平台。

传统合规服务的痛点在于：**贵、慢、不可解释**。一个合规咨询动辄数万元，一份认证报告等几周，结论依赖专家经验无法自动化。避风港的目标是将这些服务转化为 **普惠型数字化基础设施**——用 AI 驱动合规检查，用规则引擎保证可解释性，用多 Agent 架构实现自动闭环。

### 核心价值主张

| 维度 | 传统合规服务 | 避风港 |
|------|-------------|--------|
| 成本 | 单次咨询 ¥5,000-50,000 | 订阅制，月费百元级 |
| 速度 | 人工审核 3-14 天 | AI 秒级响应 |
| 覆盖面 | 单一市场/单一品类 | 4 大市场 × 全品类 |
| 主动性 | 被动等用户咨询 | 法规变更主动推送 |
| 可解释性 | 依赖专家经验 | 规则引擎 + LLM 双路径 |
| 店铺集成 | 无 | Shopify 原生 OAuth |

## 三、功能矩阵

### 3.1 合规检测核心

| 功能 | 说明 | 后端 API |
|------|------|---------|
| HS 编码匹配 | 输入产品名称/类型 → 自动匹配 6 位 HS 编码 | `GET /products/{id}/compliance` |
| VAT 税率查询 | 按 HS 编码 + 目的国查税率 | `data/vat_rates.json` |
| 认证矩阵 | CE/FCC/REACH/PSE/KC 等认证要求 | 合规规则引擎 |
| 管制品检查 | 制裁国 / 高危 HS / 新疆棉 / 稀土 | `GET /customs/controlled-goods/check` |
| 风险标记 | 低/中/高/禁运 四级风险评级 | `POST /pipeline/check` |

### 3.2 Shopify 店铺集成

| 功能 | 说明 |
|------|------|
| OAuth 授权 | CLI + Web 双模式授权，安全获取 access_token |
| 产品同步 | 定时（每 20 分钟）全量/增量同步 Shopify 商品到本地 |
| 缺失检测 | 本地无但 Shopify 有的商品 → 飞书群推送提醒 |
| 合规自动触发 | 新产品创建 Webhook → 自动触发合规检查 |
| Metafields 绑定 | HS 编码、合规状态写入 Shopify Metafields |

### 3.3 风险情报监控

| 功能 | 说明 |
|------|------|
| 多市场监控 | EU（欧盟）/ US（美国）/ JP（日本）/ KR（韩国）|
| 法规变更扫描 | 定时（每小时）扫描法规源 → AI 分析影响范围 |
| 风险等级评估 | 低 / 中 / 高 / 紧急 四级 |
| 多渠道推送 | 飞书 Bot 群消息 / Webhook 回调 |

### 3.4 多 Agent 协同体系

| Agent | 职责 |
|-------|------|
| Manager Agent | 事件总线监听 → 任务分发 → 结果汇总 |
| QA Agent | 系统健康诊断、合规质量保证 |
| Compliance Worker | 产品合规检查执行 |
| Regulation Worker | 法规变更扫描执行 |
| Listing Worker | Shopify 商品同步执行 |

### 3.5 物流报关全链路

| 功能 | 说明 |
|------|------|
| 销售订单 | CRUD + 支付记录 + 状态流转 |
| 三单一致性 | 6 维度检查：支付完整性、金额匹配、申报价值、收货人、数量、目的国 |
| 报关单生成 | 15 个扩展字段（品牌/型号/发货人/EORI/许可证等） |
| 关税计算 | 20 国税率表（含 22 个 EU 成员国） |
| 物流追踪 | 17TRACK / AfterShip Webhook 接入 |

### 3.6 RAG 知识库

| 功能 | 说明 |
|------|------|
| 文档导入 | Markdown 文件批量导入，自动分块向量化 |
| 语义检索 | ChromaDB 向量相似度检索 |
| Agent 上下文 | 合规检查时自动注入相关知识片段 |
| 知识管理 | 前端可视化管理文档库 |

## 四、用户场景

### 场景 1：新产品上架合规自查

> "我有一个蓝牙耳机要卖到欧盟，需要什么认证？"

1. 用户在对话界面输入产品信息
2. AI Agent 自动查询 HS 编码 → 匹配 CE/REACH 认证要求
3. 检索知识库中欧盟电子相关法规
4. 返回结构化合规报告：需要 CE 认证、REACH 合规声明、WEEE 回收标识

### 场景 2：Shopify 店铺自动合规

> "Shopify 店铺新增了商品，系统自动检查合规状态"

1. Shopify Webhook 触发 `product:created` 事件
2. Manager Agent 接收事件 → 分发给 Compliance Worker
3. Worker 提取产品信息 → 执行合规检查
4. 检查结果写入产品记录 → 通过飞书通知卖家

### 场景 3：法规变更预警

> "欧盟发布了新的电池法规，影响哪些产品？"

1. 定时任务（每小时）扫描欧盟法规源
2. 检测到新法规 → AI 分析影响范围
3. 匹配本地产品库中的电池/电子产品
4. 飞书群推送预警，标记受影响产品列表

### 场景 4：报关单证准备

> "100 个订单发往德国，准备报关材料"

1. 从销售订单导入商品清单
2. 运行三单一致性检查（订单/支付/物流）
3. 自动生成报关单（含 HS 编码、申报价值、EORI 等）
4. 计算关税（德国 VAT 19% + 关税）

## 六、竞争优势

1. **全链路覆盖**：从产品上架到报关物流，一站式合规管理
2. **AI + 规则双引擎**：LLM 提供智能推理，规则引擎保证可解释性和确定性
3. **Shopify 原生集成**：不是简单的 API 对接，而是通过 Claude Agent SDK + Shopify AI Toolkit 深度集成
4. **事件驱动架构**：新需求 = 新事件 + 新 Worker，扩展零侵入
5. **主动响应**：不是被动等用户查询，而是法规变更主动推送
6. **低成本普惠**：月费百元级替代万元级人工咨询
